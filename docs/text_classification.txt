# 文本分类与情感分析技术

## 文本分类概述

文本分类是NLP中最基础也是最重要的任务之一，旨在将文本自动划分到预定义的类别中。根据分类目标的不同，文本分类主要分为以下几种类型：

### 1. 二分类
判断文本属于某个类别或不属于，如垃圾邮件检测。

### 2. 多分类
将文本划分到多个互斥类别中，如新闻分类（政治、经济、体育等）。

### 3. 多标签分类
文本可以同时属于多个类别，如文章标签标注。

## 文本分类的发展历程

### 传统机器学习方法

1. **朴素贝叶斯（Naive Bayes）**
   - 基于贝叶斯定理，假设特征条件独立
   - 优点：训练和预测速度快，适合大规模文本
   - 缺点：特征独立假设过于简化

2. **支持向量机（SVM）**
   - 在高维空间寻找最优分类超平面
   - 优点：对高维稀疏特征效果好
   - 缺点：对大规模数据训练成本高

3. **逻辑回归（Logistic Regression）**
   - 使用Sigmoid函数输出分类概率
   - 优点：输出概率值，可解释性强
   - 缺点：难以处理复杂非线性关系

### 深度学习方法

1. **TextCNN**
   - 使用多个尺寸的卷积核提取n-gram特征
   - 架构：Embedding → Conv → MaxPooling → FC

2. **RCNN（循环卷积神经网络）**
   - 结合RNN和CNN的优点
   - 使用双向RNN捕获上下文信息

3. **BERT等预训练模型**
   - 在大规模语料上预训练
   - 通过微调适应下游分类任务

## 文本预处理技术

### 1. 分词（Tokenization）
- 英文：基于空格和标点
- 中文：基于词典和统计方法（Jieba、HanLP等）

### 2. 去停用词
去除高频但无意义的词（如"的"、"the"、"is"等）。

### 3. 词干提取和词形还原
- 词干提取：粗暴地截断词尾（running → run）
- 词形还原：考虑词性还原为词根形式（running → run, better → good）

### 4. n-gram特征
提取连续n个词的组合，如unigram、bigram、trigram。

## 特征表示方法

### 1. 词袋模型（Bag of Words）
将文本表示为词频向量，忽略词序。

### 2. TF-IDF
考虑词的重要性，常见词惩罚权重。

### 3. 词向量（Word Embedding）
将词映射到低维稠密向量空间。

### 4. 句子/文档向量
- Doc2Vec
- Sentence-BERT
- Universal Sentence Encoder

## 情感分析详解

情感分析是文本分类的一个重要分支，旨在识别文本中表达的情感倾向。

### 情感分析的类型

1. **粗粒度情感分析**
   - 正面/负面二分类
   - 正面/中性/负面三分类

2. **细粒度情感分析**
   - 多层级情感分析
   - 情感强度分析（如1-5星评分）

3. **方面级情感分析（ABSA）**
   - 识别文本中提到的不同方面
   - 判断每个方面的情感极性
   - 例如："这家餐厅的菜品很好吃，但服务态度很差"

### 情感分析的常用方法

1. **基于词典的方法**
   - 使用情感词典（如SentiWordNet）
   - 计算情感词得分总和

2. **基于机器学习的方法**
   - 使用SVM、朴素贝叶斯等分类器
   - 特征：词袋、TF-IDF、词向量

3. **基于深度学习的方法**
   - TextCNN、LSTM、BERT等模型
   - 端到端训练，效果最好

4. **基于Transformer的预训练模型**
   - 使用BERT、RoBERTa等模型
   - 在情感分析数据集上微调
   - 当前效果最好的方法

## 情感分析面临的挑战

1. **讽刺和反讽**
   - 表面正面的词汇可能表达负面情感
   - 例如："真是太好了，又堵车了！"

2. **情感迁移**
   - 随着时间推移，情感倾向可能改变

3. **多语言和表情符号**
   - 需要处理不同语言的情感表达
   - 表情符号往往携带重要情感信息

4. **领域适应性**
   - 不同领域的情感表达方式不同
   - 例如：电子产品评论 vs. 电影评论

## 情感分析在RAG系统中的应用

在检索增强生成系统中，情感分析可用于：

1. **用户反馈分析**：分析用户对产品的评价
2. **舆情监控**：实时监测公众对特定话题的情感态度
3. **个性化推荐**：根据情感倾向提供个性化内容
4. **对话情感识别**：在对话系统中识别用户情感状态

## 评估指标

### 分类任务常用指标

1. **准确率（Accuracy）**：正确分类的样本比例
2. **精确率（Precision）**：预测为正类中实际为正类的比例
3. **召回率（Recall）**：实际正类中被正确预测的比例
4. **F1分数**：精确率和召回率的调和平均

### 不平衡数据处理

当正负样本比例严重失衡时：
- 使用类别权重
- 过采样（SMOTE）或欠采样
- 调整分类阈值
